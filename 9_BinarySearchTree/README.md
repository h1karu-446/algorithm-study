#　二分探索木
各ノードがキーを持ち、二分探索木条件を常に満たすように構築された二分木のこと。  
各ノードはkeyに加え、親、左の子、右の子への**ポインタ**を持つ

> 二分探索木条件  
> 左部分木の全てのkey < ノードのkey  
> 右部分木の全てのkey > ノードのkey

## 二分探索木の挿入（Insert）

### 概要
根（root）から探索を開始し、挿入位置を見つけて新しいノードを追加する。

### 手順
1. 探索用ポインタを root に設定する
2. ポインタが NIL（または NULL）になるまで繰り返す
   - 挿入する値と現在ノードの値を比較
   - 小さい場合：左の子へ進む
   - 大きい場合：右の子へ進む
3. 挿入位置に到達したらノードを追加する
   - このとき、**直前のノード（親）を保持しておく必要がある**

### ポイント
- 挿入位置は「子が存在しない場所（NIL）」になる
- 親ノードを保持しておくことで、左右どちらに接続するか決定できる

### 計算量
- **O(h)**（h：木の高さ）

| ケース | 計算量 |
|--------|--------|
| 平均（バランスが良い場合） | O(log n) |
| 最悪（偏った場合） | O(n) |

### 補足
- ソート済みのデータを順に挿入すると、片側に偏った木になりO(n)になる。
- この問題を解決するために、AVL木や赤黒木などの平衡木が用いられる
