# 木構造
ノード(根)とエッジ(辺)で表されるデータ構造。階層構造に適している

## rotted tree
root(根)と呼ばれる他と区別された1つの特定の頂点を基準に階層構造を持たせた木。

- 親：あるノードの1つ上のノード
- 子：あるノードの1つ下のノード
- 次数：そのノードにおける子の数
- 兄弟：同じ親を持つノード
- 葉：子を持たないノード
- 内部ノード：葉以外のノード
- 深さ；rootからそのノードまでのedgeの数
- 高さ：そのノードから一番深い葉までの距離
- 部分木：あるノードをrootとする木
- 順序木：子に順序性がある木

rootは唯一親を持たない特別なノード
