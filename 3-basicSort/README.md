# 初等的整列
この章では初等的整列をまとめる。

## 挿入ソート
既に整列済みの配列の適切な位置に、対象要素を挿入することで整列する
- 安定
- O(n^2)
- データの並びが計算量に大きく影響する。降順の時最も遅く昇順の時最も早い

## バブルソート
要素を隣同士比較・交換しながら整列する。
- 安定
- O(n^2)
- flagを使い一度も交換がない（整列している）なら終了する。

## 選択ソート
未ソート要素の中から最小の要素を選択し、それを未ソート部分の最小の場所と交換する

- 不安定  
  次のケースを数字で昇順にソートする時不安定になる。  
  3H | 5S | 3D | 1S  
  1S | 5S | 3D | 3H  
  1S | 3D | 5S | 3H  
  1S | 3D | 3H | 5S  
- O(n^2)
