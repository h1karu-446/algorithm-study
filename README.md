# アルゴリズム学習リポジトリ

本リポジトリは、アルゴリズムおよびデータ構造の体系的理解を目的とした学習記録です。  
C++による実装と理論的整理を並行して行い、再現可能な知識の蓄積を目指しています。

---

## 目的

- 基本的なアルゴリズム・データ構造の理解深化
- C++による標準的な実装の整理
- 計算量や正当性を含めた理論的整理
- 長期的な知識基盤の構築

---

## 学習方針

本リポジトリは、アルゴリズムおよびデータ構造を体系的に理解するため、
書籍『プログラミングコンテスト攻略のためのアルゴリズムとデータ構造』
を主な参考資料として章構成を行っています。

各章ごとにディレクトリを作成し、
実装と理論整理を並行して進めています。

---

## 方針
- 各章ごとにディレクトリを作成
- 各アルゴリズムは `.cpp` として実装
- 理論・計算量・要点は `README.md` に整理
