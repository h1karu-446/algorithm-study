# 線形探索 (Linear Search)

配列の先頭から順番に要素を調べ、目的の値を探すアルゴリズム。  
データがソートされていなくても使用できるのが特徴。  
計算量は `O(n)`。

## 番兵法 (Sentinel)

探索する配列の末尾に探索対象の値（番兵）を追加することで、  
境界チェックを省略し、比較を一致判定のみで行えるようにする手法。

# 二分探索 (Binary Search)

あらかじめ昇順または降順にソートされている配列に対して使用できる探索アルゴリズム。  
探索範囲の中央の要素と探索値を比較し、その大小関係に応じて探索範囲を半分に絞り込んでいく方法。

- 継続条件:`left < right`, 更新:`right = mid`
- 継続条件:`left <= right`, 更新:`right = mid-1`
の2通りがある
